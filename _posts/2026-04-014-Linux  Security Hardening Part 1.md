# Apa Itu  Linux Security Hardening ?
Linux Security Hardening adalah proses mengamankan sistem operasi linux dengan cara mengurangi celah keamanan (attack surface), membatasi akses yang tidak diperlukan, serta mengoptimalkan konfigurasi sistem agar lebih tahan terhadap serangan siber.

Proses ini bertujuan untuk meminimalkan potensi attack vectors dan memastikan integrity, confidentiality dan availability. Hardening linux biasanya mencakup konfigurasi pengaturan sistem, penerapan patch keamanan, dan menonaktifkan layanan dan aplikasi yang tidak perlu.
